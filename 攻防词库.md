# 15. 标准 / 框架 / 知识库名词

| 类型       | 关键词                                                       |
| ---------- | ------------------------------------------------------------ |
| 攻击知识库 | MITRE ATT&CK、CAPEC、Cyber Kill Chain                        |
| 弱点/漏洞  | CWE、CVE、CVSS、CPE、NVD、CISA KEV                           |
| Web/API    | OWASP Top 10、OWASP API Top 10、OWASP ASVS、OWASP MASVS、OWASP SAMM |
| AI 安全    | OWASP Top 10 for LLM Applications、MITRE ATLAS               |
| 防御治理   | NIST CSF、NIST SP 800-53、ISO 27001、CIS Controls、MITRE D3FEND |
| 云安全     | CSA CCM、CIS Benchmarks、Kubernetes CIS Benchmark            |
| 隐私合规   | GDPR、CCPA、PIPL、Data Classification、Data Governance       |

CWE 是软件/硬件弱点枚举，CAPEC 是攻击模式枚举与分类，二者更偏“弱点/攻击模式语言”；ATT&CK 更偏真实攻击行为链；OWASP 更偏应用层风险；NIST/D3FEND 更偏治理与防御动作。
